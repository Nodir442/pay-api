Это веб-сайт, разработанный с использованием HTML, CSS, SCSS и JavaScript для создания адаптивного интерфейса, включая бургер-меню для мобильных устройств. Сайт включает три взаимосвязанные страницы, стилизованные с помощью SCSS и mixins.

Технологии
HTML: Структура страниц.
CSS/SCSS: Стилизация с использованием SCSS, включая mixins.
JavaScript: Реализация бургер-меню для мобильных устройств и другие динамические элементы.
